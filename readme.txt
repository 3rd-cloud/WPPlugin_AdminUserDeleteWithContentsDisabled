ユーザー削除時に「すべてのコンテンツを消去します。」を選択できなくします。

== Description ==

ユーザー削除時に「すべてのコンテンツを消去します。」を選択すると、削除されるユーザー所有の記事やメディアライブラリも一生に削除されます。
上記操作を知らないユーザーが誤った操作を行い、記事やメディアライブラリのファイルをまとめて削除する事故を防ぐために作成されました。

== Installation ==

1. Upload `disabled-user-delete-with-contents/plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'ユーザー削除の制限' menu in WordPress

== Frequently asked questions ==

Q. オプション設定はありますか?

A. ありません。プラグインを有効にするだけでOKです。

ユーザー削除時に「すべてのコンテンツを消去します。」を選択できなくします。

== Description ==

ユーザー削除時に「すべてのコンテンツを消去します。」を選択すると、削除されるユーザー所有の記事やメディアライブラリも一緒に削除されます。
上記操作を知らないユーザーが誤った操作を行い、記事やメディアライブラリのファイルをまとめて削除する事故を防ぐために作成されました。

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “Admin User Delete With Contents Disabled”.
3. Select the “My Custom Style Css Manager” plugin and click “Install”.
4. Activate the plugin.

OR…

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
